# Pagina personal con calculadora

Este proyecto es una pagina web personal creada con HTML, CSS y JavaScript.
Incluye una seccion de presentacion personal y una calculadora de operaciones
basicas: suma, resta, multiplicacion y division.

## Archivos del proyecto

- `index.html`: estructura de la pagina
- `style.css`: estilos visuales y diseno responsive
- `script.js`: logica de la calculadora

## Como ver el proyecto

1. Abre el archivo `index.html` en tu navegador.
2. Ingresa dos numeros en la calculadora.
3. Presiona el boton de la operacion que quieras realizar.

## Personalizacion

Antes de entregar, reemplaza estos datos en `index.html`:

- `Tu Nombre Aqui`
- `tunombre@correo.com`
- ciudad, carrera y descripcion personal

## Publicarlo en GitHub

1. Crea un repositorio nuevo en GitHub.
2. En la terminal, dentro de esta carpeta, ejecuta:

```bash
git add .
git commit -m "Agregar pagina personal con calculadora"
git branch -M main
git remote add origin https://github.com/tu-usuario/tu-repositorio.git
git push -u origin main
```

3. Comparte el enlace del repositorio como entrega.
